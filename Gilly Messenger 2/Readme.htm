  1. Introduction :-
    Gilly Messenger is a MSN Messenger client programmed in Visual Basic.
    It signs you in to MSN server and lets you do the tricks ;)

  2. What Gilly Messenger does?
    1. Multiline nick names
    2. Talk to people while they're blocked.
    3. Talk to people while you're appearing offline (You need to appear online once).
    4. Talk to people while you're signed out.
    5. Open multi sessions of Gilly Messenger.
    6. Search your contact list.
    7. Chat Logger
    8. Text Styler (Write messages in different styles).
    9. Auto message
    10. GM Scripting Engine (You can create scripts for GM using its own scripting language).
    11. Message can be as long as 1536 characters.
    12. No invite limit (You can invite as many people in your chat as you like).
    13. Commands in chat window i.e CLS, Invite
    14. New IDLE state (The person who doesn't touch messenger for 5 mins shows up as Idle)
    15. New IGNORE option
    16. Easy to change nick and status.
    17. Block alert
    18. Delete alert
    19. Stealth sign in (You're appeared offline by default. Block the unwanted people and then go online ;)
    20. Remote Control (You can operate your Messenger/PC from anywhere in the world).
    21. Store comments for each contact.
    22. Status Logger

  3. Chat window commands :-
    You can type the following commands for quick actions :-

    1. CLS
      Clears the text of your chat window.
       
    2. /Reconnect
      Reconnects to primary conversation buddy if he/she closes the window.
       
    3. /Vanish
      Hides the conversation window till a message is received.
       
    4. /Profile
      Opens the profile of the primary chat window buddy.
       
    5. /Mask
      Masks messages that you type.
       
    6. /Block
      Blocks/Unblocks the primary conversation buddy.
       
    7. /Ignore
      Ignores/Unignores the primary conversation buddy.
       
    8. /Invite EmailAddress
      Invites a person to the conversation.
       
    9. /ChatLogger
      Starts/Stops the Chat Logger.
       
    10. /List
      Displays the buddies in current conversation.
       
    11. /List Online
      Displays the buddies currently online.
       
    12. /Find Text
      Searches the chat box for specified text.
       
    13. /Nick NewNick
      Changes your nick name
       
    14. /Away Message
      Sets your status to away with an auto message.
       
    15. /Online
      Sets your status to Online
       
    16. /Busy
      Sets your status to Busy
       
    17. /Brb
      Sets your status to Be Right Back
       
    18. /Away
      Sets your status to Away
       
    19. /Phone
      Sets your status to On The Phone
       
    20. /Lunch
      Sets your status to Out To Lunch
       
    21. /Idle
      Sets your status to Idle
       
    22. /Hide
      Sets your status to Appear Offline
       
    23. /Automessage Message
      Turns the automessage ON/OFF
       
    24. /Color HexCode
      Changes the color of your message to specified color value in hex code.
       
    25. /Bold
      Changes the font style to bold.
       
    26. /Italic
      Changes the font style to italic.
       
    27. /Chat Email
      Starts a conversation with the person of that email.
       
    28. /Msg Email Message
      Sends the message to contact with that email.
       
    29. /MsgAll Message
      Sends message to all opened chat windows.
       
    30. /View Log
      Opens the chat log of the primary conversation buddy.
       
    31. /Signout
      Signs you out.
       
    32. /Sigin Username Password
      Signs you in to the server with the specified username and password.
       
    33. /Msgr
      Sets focus to the GM.
       
    34. /Exit
      Exits Gilly Messenger

  5. GM Scripting Engine :-
    This engine lets you execute a series of commands saved in a file.
    Sample scripts are provided with the software.
    Following are the commands with parameter types:-

    1. REN NewNickName
      Changes your nick name.
       
    2. GTO LineNumber
      Jumps you to the specific command in the script.
       
    3. CST StatusIndex
      Changes your satus to Online, Offline, Busy, etc.
      Status Index is a integer with value from 0 to 7
       
    4. SOT
      Signs you out of MSN server.
       
    5. SIN Username Password
      Signs you into MSN server.
       
    6. SLP Interval
      Delays the execution of the next statement in milli seconds.
       
    7. AMG Message
      Sets the Auto Message.
       
    8. MGA Message
      Sends message to all opened chat windows.
       
    9. CLG
      Starts/Stops the Chat Logger.
       
    10. BLK EmailAddress
      Blocks the person from seeing your online status.
       
    11. UBK EmailAddress
      Unblocks the person.
       
    12. MSG EmailAddress Message
      Sends the message to the user with that email address.
       
    13. SND PathOfSoundFile
      Plays the specified sound file.
       
    14. EXT
      Exits Gilly Messenger.
       
    15. END
      Ends the Script.
       
    16. VAR VariableName:Value
      Stores the value in a variable with that name.
       
    17. IF EmailAddress IS [NOT] ONLINE/BLOCKED/IGNORED
      Executes the commands inside the IF block for a true statement.
      Block Must end with "end if" statement.
       
      18. If the parameter is (INP) then user is asked for the parameter of the command at run time.
      If the parameter is VAL:VariableName then the parameter is fetched from the variable with that name.
       

  6. GM Remote Control :-
    To use the Remote Control you must start the server at the target GM.
    To start the server, cick on Tools>Remote Control on the GM main window.
    Enter the username and password and press Start Server.
    When the server is started you need to identify yourself to it using that username and password.
    First you need to send a request to GM i.e 'GM Remote Control Request' (Case sensitive)
    The server will then ask you for the username and password.
    Type the username and press CTRL+ENTER to goto new line then type the password and send it.
    Once you are logged in, you are then ready to send commands to the server.
    Following DOS commands are compatible without switches :-

    1. DIR
      Shows the files and directories.
       
    2. RD
      Removes a directory.
       
    3. MD
      Creates a new directory.
       
    4. COPY
      Copies a file from source to destination directory.
       
    5. DEL
      Deletes a file.
       
    6. REN
      Renames a file or directory.
       
    7. Drive:
      Changes the drive.
       
    8. CD
      Changes the directory.
       
      9. Other commands are :-

    9. Msgr ChatWindowCommand
      Sends a chat window command to control the messenger.
       
    10. List Online
      Sends the list of online buddies on that messenger.
       
    11. Execute File/Internet Address Parameters
      Opens/Executes the file on the server with optional paramerters.
       
    12. Dump File
      Starts putting your messages into a file. To end the file dump send a CrLf (CTRL+ENTER) and '.' in a single message.
       
    13. Read File
      Reads the contents of a file.
       
    14. Logout
      Logs you out of the server.
       
    15. Reboot
      Reboots the server PC.
       
    16. Shutdown
      Turns off the server PC.
